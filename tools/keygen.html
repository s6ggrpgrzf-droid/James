Elite signing keygen

Generates an ECDSA P-256 keypair locally in your browser. Nothing is sent anywhere — close the tab and the keys are gone forever. Use this once when wiring up the Cloudflare verification Worker.

Safe to use offline. Run this page on any device — the WebCrypto API generates the keys on-device. The page makes zero network requests after it loads.

What to do with the output:

1. Copy the public key → paste into app.html at line ~124 (replace the REPLACE_ME values), commit.

2. Copy the private key → in the Cloudflare Workers dashboard for your Worker, go to Settings → Variables and Secrets → Add → Secret, name it SIGNING_PRIVATE_JWK, paste, save.

3. Close this tab. The private key only exists where you pasted it.

Generated entirely on-device using the Web Crypto API. ECDSA P-256, used to sign Elite unlock tokens.
